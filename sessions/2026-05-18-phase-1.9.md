## Decisions made

- **Per-line / poem alignment and freeform drag-to-position deferred to Phase 2+** · *Clover proposed, Claude approved* — original 1.9 scope stayed as the two-button empty-line warning. Alignment is an artistic / display concern, not a publish-time data hygiene issue; freeform pixel positioning is a data model change requiring the Phase 2.1 editor rethink. Backlog entry added with the full rationale.
- **"Remember my choice" saves to localStorage for all users; additionally fires `/me/preferences` for logged-in users** · *design call* — avoids touching the `/me/sync` flow (which coerces all values to strings, which would corrupt a boolean pref). Preference is not back-synced from anonymous→account on login; first explicit choice while logged in becomes the account pref.
