## Decisions made

- **Debounce timer set to 7s** — spec said 5–10s; 7s is in range, felt right for drag repositioning cadence
- **`autoSave()` only fires for `isLoggedIn && !isAdmin`** — admin stable is server-synced via separate endpoints; anon users continue using `saveDraftLocal()`
- **Draft auto-name "untitled"** — spec language; simple and clear while the user hasn't named it
- **Page-load picker from server-rendered `initial_drafts_json`** — avoids a client round trip on load; "Change Draft" fetches fresh list since the user may have created drafts in another tab
- **Anon localStorage check before showing picker** — prevents picker and `syncLocalToAccount()` from both running simultaneously if the user has an anon draft on first login
